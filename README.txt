הוראות לפריסה ב-Render:
1. עבור אל https://render.com
2. לחץ "New" > "Web Service"
3. בחר בפריסה דרך Git (לא קיימת כרגע פריסה ישירה מקובץ ZIP)
4. העלה את הקבצים לריפו ב-GitHub
5. הדבק את הקישור לריפו בתוך שורת ה-Repository URL ב-Render
6. ודא שיש לך שני שירותים: backend (Python) ו-frontend (Static)
